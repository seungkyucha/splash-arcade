# 💧 스플래시 아케이드 (Splash Arcade)

크레이지 아케이드 스타일의 **물풍선 배틀 웹 게임**입니다.
순수 HTML5 Canvas + Vanilla JS로 제작되었으며, 모든 그래픽과 사운드는
코드로 직접 생성한 **오리지널 에셋**입니다 (외부 에셋/원작 리소스 미사용).

## 🎮 플레이 방법

| 조작 | 키 |
|------|----|
| 이동 | 방향키 / WASD |
| 물풍선 설치 | Space |
| 바늘 사용 (갇혔을 때 탈출) | Space |
| 음소거 | M |

- AI 봇 3명과 서바이벌 대결, **최후의 1인**이 되면 승리
- 물줄기에 닿으면 물방울에 갇히고, 시간이 지나거나 적이 닿으면 사망
- 덤불을 부수면 아이템 등장

## 🧪 아이템

| 아이템 | 효과 |
|--------|------|
| 💧 물풍선 | 동시 설치 개수 +1 |
| 🧪 물약 | 물줄기 길이 +1 |
| ⭐ 울트라 물약 | 물줄기 길이 최대 |
| 👟 롤러스케이트 | 이동 속도 +1 |
| 😈 빨간 악마 | 이동 속도 최대 |
| 📍 바늘 | 갇혔을 때 탈출 (소지) |

## 🛠 기술

- HTML5 Canvas + ES Modules, 빌드 스텝 없음
- 스프라이트: 캔버스 프로시저럴 드로잉 (프리렌더)
- 사운드: WebAudio 합성 (효과음 + BGM)
- AI: BFS 기반 위험 회피 / 공격 / 아이템 수집

## 🚀 실행

```bash
npx serve .        # http://localhost:3000
node test/smoke.mjs  # 게임 로직 스모크 테스트
```
