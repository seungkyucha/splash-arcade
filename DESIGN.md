# Splash Arcade (스플래시 아케이드) — 기획 문서

크레이지 아케이드 BnB의 게임 메커니즘을 충실히 재현한 웹 게임.
그래픽/사운드는 원작의 스타일(밝은 파스텔, SD 캐릭터, 타일 맵)을 따르는 **오리지널 에셋**
(캔버스 프로시저럴 드로잉 + WebAudio 합성음)으로 제작한다. 원작 추출 에셋은 사용하지 않는다.

## 1. 원작 조사 요약

- 장르: 봄버맨류 액션 퍼즐 (넥슨, 2001)
- 맵: 15×13 타일(195칸), 파괴 가능한 상자 블록 + 파괴 불가 블록
- 핵심 룰:
  - 물풍선을 설치하면 일정 시간 후 폭발, 십자(상하좌우) 방향으로 물줄기 발사
  - 물줄기는 블록에 막히며, 파괴 가능한 블록은 1개 부수고 멈춤
  - 물줄기가 다른 물풍선에 닿으면 **연쇄 폭발**
  - 물줄기에 닿은 플레이어는 **물방울에 갇힘** → 매우 느리게 이동, 풍선 설치 불가
  - 갇힌 상태에서 시간이 지나면 터져서 사망, **적이 닿으면 즉시 사망**
  - 같은 편이 닿으면 구출 (팀전), 바늘 아이템으로 즉시 탈출 가능
- 아이템 (블록 파괴 시 드랍):
  - 물풍선: 동시 설치 개수 +1
  - 물약: 물줄기 길이 +1
  - 울트라 물약: 물줄기 길이 최대
  - 롤러스케이트: 이동 속도 +1
  - 빨간 악마: 이동 속도 최대
  - 바늘: 갇혔을 때 탈출 (소지형)
- 비주얼: 머리 큰 2등신 SD 캐릭터, 밝은 채도 높은 파스텔, 탑다운 3/4 뷰 타일

## 2. 본작 사양

### 모드
- 싱글 플레이 서바이벌: 플레이어 1 + AI 봇 3, 최후의 1인 승리
- 라운드 제한 시간 120초 (시간 초과 시 무승부)

### 캐릭터 (오리지널)
| 이름 | 컨셉 | 색 |
|------|------|----|
| 코코 | 고양이 | 주황 |
| 보리 | 강아지 | 갈색/크림 |
| 피노 | 펭귄 | 남색 |
| 루루 | 토끼 | 핑크 |

### 조작
- 이동: 방향키 / WASD
- 물풍선 설치: Space
- 바늘 사용(갇혔을 때): Space 또는 자동

### 수치
- 타일: 48px, 맵 15×13
- 초기 능력치: 풍선 1개, 물줄기 1, 속도 1 (최대: 풍선 6, 물줄기 7, 속도 5)
- 물풍선 폭발 대기: 3초, 물줄기 지속: 0.45초
- 갇힘 지속: 7초 후 사망
- 아이템 드랍률: 45% (물풍선 30 / 물약 30 / 롤러 20 / 바늘 12 / 울트라 4 / 악마 4 가중치)

### AI
- 위험 감지: 설치된 풍선의 예상 폭발 범위 안이면 BFS로 안전 타일 탈출
- 공격: 적/상자가 사거리 내면 풍선 설치 (탈출로 확보 시에만)
- 탐색: 가까운 아이템 > 상자 > 적 순으로 BFS 이동
- 갇힌 적 발견 시 접근해서 처치

### 화면 구성
1. 타이틀 → 2. 캐릭터 선택 → 3. 인게임(HUD: 시간, 능력치, 바늘 수) → 4. 승리/패배 → 재시작

### 기술
- 순수 HTML5 Canvas + Vanilla JS(ES Modules), 빌드 스텝 없음 → Vercel 정적 배포
- 스프라이트: 오프스크린 캔버스에 프리렌더한 프로시저럴 픽셀아트
- 사운드: WebAudio 합성 (효과음 + 간단한 BGM 루프)
